本文将持续整理更新关于phpstorm的实用功能和快捷键。<!-- more -->
#### 1. 查看当前文件函数列表
```
Alt + 7
```
#### 2. 在当前类文件中查找方法
```
Ctrl + F12
```
#### 3. 文件名查找
```
Ctrl + Shift + N
```
#### 4. 类名查找
```
Ctrl + N
```
#### 5. 跳转至当前文件所在磁盘上的位置
```
Ctrl + Alt + F12
```
#### 6. 对比文件
```
Ctrl + 鼠标左键 选中文件，鼠标右键选择Compare Files（Ctrl + D）
```
